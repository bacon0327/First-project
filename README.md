# First-project

本專案為語音控制與 BERT 指令分類整合應用，支援語音下棋與簡易家具語句辨識。

---

## 📁 專案結構

- `main.py`：主流程整合 Whisper 與 BERT
- `tkinter_v1.py`：五子棋介面與輸出
- `bert_command_classifier.py`：BERT 指令分類模組
- `ai_gomoku.py`：AI 對弈邏輯

---

## 🔧 安裝環境建議

建議使用 Conda 建立虛擬環境，Python 版本建議為 3.10。
